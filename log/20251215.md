## two types of storage

* heap: a global area where the memory storing values bound to names are tracked

* stack; a list where function call arguments are pushed and return values popped

## time complexity

| complexity | 1 second | 1 minute | 1 hour | gain |
| ---------- | -------- | -------- | ------ | ---- |
| n, linear | 1000 | 60,000 | 3,600,000 | x60 |
| nlogn, logarithmic | 140 | 4, 893 | 200,000 | x41 |
| n^2, quadratic | 31 | 244 | 1,897 | x8 |
| n^3, cubic | 10 | 39 | 153 | x4 |
| 2^n, exponential | 9 | 15 | 21 | +6 |
