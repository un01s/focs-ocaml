# foundations of computer science in ocaml

## [install ocaml](https://ocaml.org/docs/installing-ocaml)

* for macOS, it is simple:

```
$brew install opam
$opam init -y
$opam install ocaml-lsp-server odoc ocamlformat utop
```

* check installation

```
$utop
```
